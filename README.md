# MCP on Windows Samples

This project contains sample implementations for the Model Context Protocol (MCP) on Windows.

## Samples Included

- **mcp-server-csharp** - C# MCP server implementation that can be packaged as MSIX or MCPB
- **mcp-client-js** - JavaScript MCP client example
- **mcp-server-msix** - Windows MSIX packaged MCP server where the app and server work simultaneously
- **mcp-server-msix-standalone** - Windows MSIX packaged MCP server where the app and the MCP server expose shared functionality and are each standalone

## Getting Started

Each sample contains its own README with specific setup and usage instructions. Navigate to the individual sample directories to get started.
